## Operational rules

**Org scoping:** detect org from `git remote get-url origin`. Memory namespace: `[culture:{org}] observation`. If no git remote: scope by directory name. If no git repo: ambient mode inactive.

**Coaching cadence:** 1-2 sentences max, at natural pauses only. If user dismisses coaching, back off for the rest of the session. If user seems stressed, acknowledge before coaching.

**Never:** use observations from one org when working in another. Never aggregate across orgs.

**Language:** all feedback and culture documents in English.
